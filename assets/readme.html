Welcome to Yafi - an RSS Reader for Yahoo! Finance

If you're following the financial market it's crutial to keep up-to-date with the latest development.

Subscribing Yahoo! Finance RSS feed is one way to ensure that you're doing just that. If you have been following Yahoo! Finance RSS feeds on your desktop Yafi will be a natural extension for your mobile needs.

Actionbar Quick Help
Sync all your feeds
Display available topics to which you can subscribe. Those topics you select will be displayed in the main UI as a separate tab.
Perform a search against a local database

Main Menu Quick Help
Retention Policy
 Check to delete RSS feeds older than 30 days
Sync at Startup
 Check to automatically pull RSS feeds from server when the app starts up

Yafi provides these:

  • Subscribe multiple topics provided by Yahoo! Financial RSS feeds
  • Sync all RSS subscriptions with a one-click of a button
  • Click on any of the feeds to follow up in the Internet
  • Store Yahoo! Finance RSS feeds on a local database for which you can search
  • Automatically clean up older RSS feeds to keep the size of a local database minimal

Yafi is short for Yahoo! Finance. Yafi grew out of a project I worked on for school. If you like this app please add a favorable rating and comment on Android Market. It will encourage me to update and provide fixes.
